## Demo 1: Capacity Metrics Comparison

### Setup Required
- Access to Fabric capacity metrics app
- Sample capacity data (or use screenshots)

### Demo Script
1. Open the Microsoft Fabric Capacity Metrics app
2. Show the CU consumption over time
3. Explain what operations consume CUs
4. Compare to equivalent Power BI Pro license count

### Key Points to Highlight
- CU consumption varies by workload
- Some operations are more expensive than others
- Burst vs. smoothed consumption

## Demo 2: Cost Calculator Walkthrough

### Setup Required
- Azure pricing calculator open
- Pre-configured scenarios (optional)

### Demo Script
1. Open [Azure Pricing Calculator](https://azure.microsoft.com/pricing/calculator/)
2. Add Microsoft Fabric
3. Walk through SKU options
4. Model scenario: "50 users, 10GB data, daily refresh"
5. Compare to: "50 Power BI Pro licenses"

### Key Points to Highlight
- Break-even analysis
- Hidden costs (storage, egress)
- Value beyond just BI (Lakehouse, Pipelines, etc.)

## Demo Tips

- Have backup screenshots in case of connectivity issues
- Prepare specific numbers for your audience's scale
